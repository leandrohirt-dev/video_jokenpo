# Jogo: Pedra, Papel e Tesoura (Jokenpô)

Este é um jogo simples e divertido de Pedra, Papel e Tesoura (também conhecido como Jokenpô), implementado em Python. O jogador joga contra o computador, que toma suas decisões de forma aleatória.

## Como Executar

1. Certifique-se de ter o Python instalado na sua máquina (versão 3.6 ou superior recomendada).
2. Clone o repositório ou baixe os arquivos.
3. Abra um terminal ou prompt de comando na pasta do projeto.
4. Execute o seguinte comando:

```bash
python main.py
```

## Como Jogar

1. Ao executar o script, será solicitado que você escolha entre: `pedra`, `papel` ou `tesoura`.
2. Digite sua escolha e pressione `Enter`.
3. O computador fará a escolha dele aleatoriamente e o resultado da rodada será exibido na tela (Vitória, Derrota ou Empate).

## Tecnologias

- Python 3
- Biblioteca `random`
- Biblioteca `time`
